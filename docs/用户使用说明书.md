## 1. 系统简介

**啄木鸟心理预警辅助系统**是一套面向学校心理健康工作的综合管理平台，涵盖量表测评、预警识别、访谈管理、随访跟踪等功能，帮助心理老师和班主任系统化地开展学生心理健康工作。

### 核心功能

| 功能模块 | 说明 |
|---------|------|
| 量表管理 | 管理自定义量表和种子量表库 |
| 测评任务 | 创建、发布、管理测评任务 |
| 学生答题 | 学生在线作答量表 |
| 测评结果 | 查看/统计/导出测评结果 |
| 预警系统 | 自动触发预警，支持处理和随访 |
| 访谈管理 | 访谈记录、文件上传、OCR识别、摘要提取 |
| 随访管理 | 跟踪需关注学生的后续状态 |
| 学生档案 | 查看学生完整心理档案（时间线视图） |
| 系统设置 | 年级/班级/学生/角色/用户/插件管理 |
| 数据库备份 | 备份与恢复系统数据 |
| 数据看板 | 综合统计概览 |
| 数据导出 | 导出Excel报表 |

### 系统架构

本系统采用桌面客户端方式运行，启动后自动在本地启动后端服务并打开浏览器界面。

## 2. 登录与角色说明

### 2.1 登录步骤

1. 双击桌面上的啄木鸟图标或运行启动脚本
2. 系统自动打开浏览器，显示登录页面
3. 输入**用户名**和**密码**
4. 点击**"登录"**按钮
5. 系统根据您的角色自动跳转到对应界面

### 2.2 角色说明

| 角色 | 说明 | 可访问界面 |
|------|------|-----------|
| **管理员** (admin) | 系统最高权限，管理所有功能 | 管理员界面（含全部设置功能） |
| **心理老师** (psychologist) | 负责心理测评、预警处理、访谈管理 | 教师界面 + 管理员界面 |
| **班主任** (teacher) | 查看本班学生测评结果，协助预警处理 | 教师界面 |
| **学生** (student) | 答题、查看自己的测评结果 | 学生界面 |

> **注意**: 心理老师角色同时拥有教师界面和管理员界面的访问权限，可以完成所有心理测评相关工作以及系统管理工作。

## 3. 工作流A: 测评预警工作流

> **流程**: 量表准备 → 创建任务 → 学生答题 → 查看结果 → 预警处理 → 访谈 → 随访 → 归档

这是系统最核心的工作流，覆盖从心理测评到后续跟进的完整流程。

### 3.1 量表管理

#### 3.1.1 浏览种子量表库

种子量表是系统预设的标准心理量表，可以直接选用。

**操作步骤**:

1. 在侧边栏点击 **"量表管理" > "种子量表"**
2. 浏览系统预置的量表列表（显示名称、版本、题目数、描述）
3. 点击某量表的 **"预览"** 按钮，查看详细信息：
   - 量表版本和描述
   - 完整题目列表和选项
   - 各分数区间的解释
4. 点击 **"选用"** 按钮，将量表克隆到自定义量表中
5. 系统自动跳转到编辑页面，您可以根据需要修改

#### 3.1.2 管理自定义量表

**操作步骤**:

1. 在侧边栏点击 **"量表管理" > "自定义量表"**
2. 查看已有的自定义量表列表
3. 点击 **"导入量表"** 按钮可以上传量表文件（JSON格式）
4. 点击 **"编辑"** 进入量表编辑页面

#### 3.1.3 编辑量表

在量表编辑页面，您可以自定义量表的各个部分：

**基本信息的填写**:
- **量表名称**: 输入量表的名称（如"中学生心理健康量表"）
- **版本号**: 输入版本号
- **描述**: 输入量表的用途说明

**维度定义**（可选）:
- 在"维度定义"区域输入维度名称（如"焦虑"、"抑郁"、"人际交往"）
- 按回车或点击"添加"按钮添加维度
- 最多可添加100个维度
- 维度用于对题目进行分类统计

**题目编辑**:
- 每道题显示为一张卡片，包含：
  - **题目内容**: 在文本框中输入题目文字
  - **题目类型**: 选择"单选"、"多选"或"填空"
  - **反向计分**: 如果该题是反向题，打开此开关
  - **所属维度**: 从下拉框选择该题属于哪个维度（可选）
  - **选项列表**: 为每个选项输入文本和对应分值
- 点击题目下方的 **"添加选项"** 增加新选项
- 点击题目右上角的 **"删除"** 移除该题
- 点击底部的 **"添加题目"** 按钮新增一道题

**保存**: 编辑完成后点击页面底部的 **"保存"** 按钮。

### 3.2 创建测评任务

#### 3.2.1 创建任务

1. 在侧边栏点击 **"测评任务"**
2. 点击 **"创建任务"** 按钮
3. 在弹出的对话框中填写：
   - **标题**: 输入任务标题（如"2026年春季学期心理健康普查"）
   - **量表**: 从下拉框选择要使用的量表
   - **目标班级**: 选择参加测评的班级（支持多选）
   - **截止时间**: 设置答题截止日期
4. 点击 **"确定"** 创建任务（初始状态为"草稿"）

#### 3.2.2 发布任务

1. 在任务列表中找到刚创建的任务（状态显示为蓝色"草稿"标签）
2. 点击 **"发布"** 按钮
3. 任务发布后，选中的班级学生将能看到该任务并开始答题

#### 3.2.3 结束任务

1. 学生答题完成后，在任务列表中找到该任务（状态显示为绿色"已发布"标签）
2. 点击 **"完成"** 按钮
3. 系统将自动计算结果并生成预警

#### 3.2.4 导出数据

点击任务列表中的 **"导出"** 按钮，可将该任务的答题数据导出为Excel文件。

### 3.3 学生答题

> 本节面向**学生**用户。

#### 3.3.1 签署知情同意书

首次进行测评前，学生需要签署知情同意书：

1. 登录后，在侧边栏点击 **"知情同意"**
2. 仔细阅读知情同意书内容
3. 勾选 **"我已阅读并同意上述知情同意书"**
4. （可选）在补充说明栏填写意见
5. 点击 **"签署同意"** 按钮

> 知情同意书签署后，系统会使用SHA-256哈希算法对签署内容进行存证。

#### 3.3.2 完成答题

1. 登录后，在侧边栏点击 **"我的任务"**
2. 查看待完成的测评任务列表
3. 点击任务的 **"作答"** 按钮
4. 进入答题页面后，逐题阅读题目并选择答案
5. 每道题必须作答，系统会在提交前检查
6. 全部作答完成后，点击页面底部的 **"提交答案"** 按钮
7. 提交成功后自动返回任务列表

#### 3.3.3 查看我的结果

1. 在侧边栏点击 **"我的结果"**
2. 查看历次测评的结果列表，包括：
   - 总分
   - 等级评估
   - 风险颜色标识
   - 专业建议
   - 测评时间

### 3.4 查看测评结果

> 本节面向**心理老师**和**班主任**。

#### 3.4.1 查看全部结果

1. 在侧边栏点击 **"结果查看"**
2. 查看所有学生的测评结果列表，包括：
   - 学生姓名、班级
   - 使用量表
   - 总分、等级
   - 风险颜色（红色=高风险，黄色=中风险，绿色=正常）
   - 专业建议
3. 点击 **"查看详情"** 查看某学生的完整测评报告

#### 3.4.2 按班级/年级筛选

1. 在结果查看页面顶部，使用 **班级下拉框** 或 **年级下拉框** 筛选
2. 选择后自动跳转到对应班级/年级的结果汇总页面
3. 班级/年级结果页面支持 **"导出Excel"**

#### 3.4.3 查看单条结果详情

1. 在结果列表中点击 **"查看详情"**
2. 查看学生的基本信息（姓名、班级、量表、任务等）
3. 查看总分和风险等级评估
4. 查看各维度的详细得分
5. 点击 **"返回"** 回到列表

### 3.5 预警处理

系统根据测评结果自动生成预警，分为以下等级：

| 预警等级 | 颜色标识 | 说明 |
|---------|---------|------|
| 红色预警 | 红色 | 需要立即关注和处理 |
| 黄色预警 | 黄色 | 需要关注，安排后续跟进 |
| 正常 | 绿色 | 无需特别关注 |

#### 3.5.1 查看预警

1. 通过数据看板（见第6章）查看预警统计概览
2. 预警列表中每条记录显示：
   - 学生姓名（可点击查看详情）
   - 预警等级（红/黄/绿标签）
   - 处理状态（待处理/已处理/已随访）
   - 预警时间

#### 3.5.2 处理预警

1. 找到状态为"待处理"的预警记录
2. 点击 **"处理"** 按钮
3. 在弹出框中填写处理备注（必填）
4. 点击确定提交
5. 预警状态变为"已处理"

#### 3.5.3 记录随访

1. 对已处理的预警，点击 **"随访"** 按钮
2. 在弹出框中填写随访备注（必填）
3. 点击确定提交
4. 预警状态变为"已随访"

#### 3.5.4 查看预警详情

1. 点击学生姓名或 **"查看"** 按钮
2. 弹出预警详情对话框：
   - 显示学生信息和预警详情
   - 显示处理记录时间线（包含每次处理/随访的时间和备注）
3. 点击 **"查看档案"** 可跳转到该学生的完整档案页

### 3.6 访谈与随访

#### 3.6.1 随访管理

1. 在侧边栏点击 **"访谈管理" > "随访管理"**
2. 设置 **风险阈值** 筛选条件：
   - "黄色及以上": 显示所有黄色和红色预警的学生
   - "仅红色": 只显示红色预警的学生
3. 系统自动列出符合条件的学生列表，包括：
   - 学生姓名、班级
   - 风险等级和颜色标签
   - 访谈次数、最近访谈时间
   - 随访状态（待随访/已访谈）
4. 点击 **"查看详情"** 进入学生档案页面

### 3.7 学生档案

学生档案是每个学生的完整心理健康记录，以时间线形式展示。

#### 3.7.1 进入学生档案

从以下入口可以进入学生档案：
- 预警列表 → 点击"查看档案"
- 随访管理 → 点击"查看详情"

#### 3.7.2 查看档案内容

学生档案页面包含以下区域：

**基本信息区**:
- 学生姓名、年级、班级、性别
- 当前风险等级（红/黄/绿标签）
- 最近测评日期
- 建议调整标签

**事件时间线**（最多显示最近20条）:
- 预警事件（红色/金色/绿色圆点）: 显示预警等级和处理记录
- 测评事件（蓝色圆点）: 显示量表测评和结果
- 访谈事件（紫色圆点）: 显示访谈记录和状态

**详细记录区**:
- 预警记录表格: 等级、状态、时间
- 测评历史表格: 结果颜色、时间
- 访谈记录表格: 类型、状态、时间，可点击 **"查看"** 跳转访谈详情

## 4. 工作流B: 访谈记录工作流

> **流程**: 访谈模板 → 创建访谈 → 上传文件 → OCR识别 → 提取摘要 → 学生档案归档

### 4.1 访谈模板管理

访谈模板定义了访谈记录需要提取的结构化字段。

#### 4.1.1 创建模板

1. 在侧边栏点击 **"访谈管理" > "模板管理"**
2. 点击 **"新建模板"** 按钮
3. 填写：
   - **模板名称**: 如"学生心理访谈记录模板"
   - **描述**: 模板用途说明
   - **字段定义(JSON)**: 输入结构化字段，格式示例：
     ```json
     [
       {"key": "performance", "label": "表现评价"},
       {"key": "suggestion", "label": "建议措施"},
       {"key": "followup_plan", "label": "后续计划"}
     ]
     ```
4. 点击确定创建

#### 4.1.2 编辑/删除模板

- 点击模板列表中的 **"编辑"** 修改模板
- 点击 **"删除"** 移除不需要的模板（需确认）

### 4.2 创建访谈记录

#### 4.2.1 新建访谈

1. 在侧边栏点击 **"访谈管理" > "访谈列表"**
2. 点击 **"新建访谈"** 按钮
3. 填写访谈信息：
   - **学生**: 从下拉框选择学生（必填）
   - **心理老师**: 从下拉框选择访谈老师（必填）
   - **访谈日期**: 选择日期和时间（必填）
   - **地点**: 输入访谈地点
   - **风险等级**: 选择"一般/关注/重点/危机"（必填）
   - **访谈模板**: 选择使用的模板（可选）
   - **备注**: 填写补充说明
4. 点击确定创建（初始状态为"草稿"）

#### 4.2.2 访谈状态流转

访谈记录有三种状态：

| 状态 | 颜色标签 | 说明 |
|------|---------|------|
| 草稿 | 蓝色 | 新创建，可编辑 |
| 已审阅 | 橙色 | 已完成初步审阅 |
| 已完成 | 绿色 | 访谈全流程完成，归档 |

状态推进: 草稿 → 已审阅 → 已完成

### 4.3 上传访谈文件

1. 在访谈列表中点击 **"查看"** 进入访谈详情
2. 切换到 **"文件管理"** 标签页
3. 将文件拖拽到上传区域，或点击选择文件
4. 支持的文件格式: `.jpg` `.jpeg` `.png` `.pdf`
5. 上传后系统自动触发OCR文字识别
6. 在文件列表中可以查看：
   - 文件名和上传时间
   - OCR处理状态（待处理/已完成/失败）
7. 如OCR失败，可点击 **"重新OCR"**
8. 可删除不需要的文件

### 4.4 OCR文字识别

1. 在访谈详情中切换到 **"OCR文本"** 标签页
2. 查看系统自动识别的文本内容
3. 如有识别错误，可以直接在文本框中手动修正
4. 修改后点击 **"保存"** 保存修正内容
5. 如需重新识别，点击 **"重新OCR"** 按钮

### 4.5 提取访谈摘要

1. 在访谈详情中切换到 **"结构化摘要"** 标签页
2. 系统会根据所选模板自动从OCR文本中提取结构化字段
3. 如自动提取未触发，点击 **"手动提取摘要"** 或 **"重新提取"**
4. 查看各字段的提取结果，手动修正不准确的内容
5. 展开 **"原始OCR文本"** 面板可对照原文检查
6. 确认无误后点击 **"保存摘要"**
7. 回到"基本信息"标签页，点击 **"标记为已审阅"** → **"标记为已完成"** 完成归档

### 4.6 查看学生时间线

1. 在侧边栏访问学生时间线页面（通过学生档案进入）
2. 以时间线形式查看该学生的所有事件：
   - 访谈事件（蓝色，文件图标）
   - 评估事件（绿色，表单图标）
   - 预警事件（红色，警告图标）
3. 每个事件显示日期和摘要信息

## 5. 工作流C: 系统管理工作流

> **流程**: 系统设置 → 年级管理 → 班级管理 → 学生管理 → 用户管理 → 数据库备份

> 系统设置功能主要面向**系统管理员**。心理老师也可访问部分管理功能。

### 5.1 年级管理

1. 在侧边栏点击 **"系统设置"**
2. 如需管理年级，通过管理员界面访问年级管理功能
3. 点击 **"新建年级"** 按钮
4. 输入年级名称（如"高一年级"）和年份（如"2026"）
5. 点击确定创建
6. 可删除不需要的年级

### 5.2 班级管理

1. 在系统设置页面，点击 **"班级管理"** 标签页
2. 点击 **"新建班级"** 按钮
3. 输入班级名称
4. 选择所属年级
5. 点击确定创建
6. 可删除不需要的班级

> **注意**: 创建班级前请先确保对应的年级已创建。

### 5.3 学生管理

#### 5.3.1 单个添加学生

1. 在系统设置页面，点击 **"学生管理"** 标签页
2. 点击 **"新建学生"** 按钮
3. 输入学生姓名、学号
4. 选择所属班级
5. 点击确定创建

#### 5.3.2 批量导入学生

1. 点击 **"下载模板"** 按钮，获取Excel导入模板
2. 按模板格式填写学生信息（姓名、学号、班级等）
3. 点击 **"导入学生"** 按钮
4. 选择填写好的Excel文件
5. 点击 **"开始导入"**
6. 系统显示导入结果：
   - 总计处理数量
   - 成功新建数量
   - 跳过数量（已存在）
   - 错误明细（如有）

### 5.4 角色管理

1. 在系统设置页面，点击 **"角色管理"** 标签页
2. 点击 **"新建角色"** 创建新角色
   - 输入角色名称和描述
3. 点击角色的 **"权限"** 按钮
4. 在穿梭框中将需要的权限从左侧移到右侧
5. 点击确定保存权限配置

### 5.5 用户管理

#### 5.5.1 创建用户

1. 在系统设置页面，点击 **"用户管理"** 标签页
2. 点击 **"新建用户"** 按钮
3. 填写：
   - **用户名**: 登录用的账号名
   - **密码**: 至少8位
   - **显示名**: 在系统中显示的名称
   - **角色**: 从下拉框选择（支持多选）
4. 点击确定创建

#### 5.5.2 编辑用户

1. 在用户列表中点击 **"编辑"** 按钮
2. 修改显示名或角色
3. 点击确定保存

### 5.6 插件管理

1. 在系统设置页面，点击 **"插件管理"** 标签页
2. 查看已安装的插件列表（名称、版本、状态）
3. 点击 **"启用"** 或 **"禁用"** 切换插件状态
4. 点击 **"配置"** 修改插件参数（仅部分插件支持）
5. 根据插件提供的表单修改参数后保存

### 5.7 数据库备份与恢复

#### 5.7.1 创建备份

1. 在系统设置页面，点击 **"数据库备份"** 标签页
2. 点击 **"创建备份"** 按钮
3. （可选）输入备份名称，或留空使用自动生成的名称
4. 点击确定，系统开始创建备份
5. 备份完成后在列表中显示文件名、大小和创建时间

#### 5.7.2 恢复数据

1. 在备份列表中找到需要恢复的备份
2. 点击 **"恢复"** 按钮
3. 系统弹出警告提示: "此操作将覆盖当前数据库，请谨慎操作！"
4. 确认后系统开始恢复数据

> **警告**: 数据恢复将覆盖当前所有数据，操作前请确保已做好必要的备份。

#### 5.7.3 删除备份

1. 在备份列表中点击 **"删除"** 按钮
2. 确认删除不需要的历史备份

## 7. 常见问题FAQ

### Q1: 学生看不到测评任务？

**可能原因**:
1. 任务尚未发布 — 请教师检查任务状态是否为"已发布"
2. 学生不在任务的目标班级中 — 检查任务的目标班级设置
3. 学生未签署知情同意书 — 部分任务要求先签署知情同意书

### Q2: 上传的访谈文件OCR识别失败？

**解决方法**:
1. 确认文件格式为 `.jpg` `.jpeg` `.png` `.pdf`
2. 检查文件是否清晰可读
3. 点击 **"重新OCR"** 重新尝试识别
4. 如仍失败，可手动在OCR文本标签页中输入内容

### Q3: 如何修改已提交的测评结果？

测评结果一旦提交不可修改。如需重新测评，请创建新的测评任务。

### Q4: 班级/年级管理中看不到某些选项？

请确认：
1. 年级已正确创建（班级需要关联年级）
2. 您的角色拥有相应管理权限（部分功能仅管理员可用）

### Q5: 数据库备份文件存储在哪里？

备份文件存储在系统数据目录中。建议定期创建备份并妥善保管。

### Q6: 如何批量导入学生？

1. 在系统设置的"学生管理"中点击"下载模板"
2. 按Excel模板格式填写学生信息
3. 点击"导入学生"上传文件
4. 系统会自动处理并显示导入结果

### Q7: 心理老师和班主任看到的界面有什么区别？

心理老师拥有完整的系统功能权限，包括：
- 所有测评、预警、访谈、随访功能
- 系统管理功能（年级/班级/学生/用户等）

班主任只能使用教师端功能：
- 查看测评结果
- 处理预警
- 查看学生档案
- 无法进行系统管理操作

### Q8: 如何更新系统版本？

系统支持OTA在线更新。当有新版本发布时，系统会自动检测并提示更新。更新过程中请勿关闭程序。

### Q9: 忘记密码怎么办？

请联系系统管理员重置密码。管理员在"用户管理"中可以删除旧用户并重新创建。

### Q10: 预警等级是如何确定的？

预警等级根据量表测评结果自动计算：
- 量表中定义了各分数区间对应的等级
- 系统根据学生的测评总分自动匹配对应的预警等级
- 红/黄/绿三色分别对应不同风险级别
